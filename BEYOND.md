# Above and Beyond Features

For this project, I added several extra features to improve the appearance and add a little spice when you win the game.

## 1. Custom Visual Design (CSS Styling)
I created a bubbly and aesthetic theme using pastel colors. I rounded different sections and used divs in my html to seperate those. Instead of everything being in one box, I separated the page into multiple smaller bubbles to make it look more modern and visually appealing.

I also styled:
- Buttons with rounded edges and hover effects
- Input boxes with softer borders and better spacing
- The overall layout so it feels clean and easy to read
- It also looks really cute
- I put the bubbles side by side and chnaged the font


## 2. Layout Improvements
I organized the page into sections that are placed both vertically and side by side. This makes the game feel less basic and more interactive instead of everything being stacked in one line. I thought this added to the game since it was in a very standard format originally. I also previously really enjoyed working in css so I wanted to do it again here.

## 3. Confetti Animation on Win
I added a confetti effect that appears when the player guesses the correct number. This uses JavaScript to dynamically create small pieces that fall down the screen. It falls down the screen like a line and a couple of cubes. It took me a while to figure how to do this and I think this might be the best I could get. I would like to practice this more because it added a lot to my project. Definitly looks a lot cuter with this.

This feature is located in:
- The `celebrateWin()` function in script.js
- Called inside the correct guess section of `makeGuess()`
- it is also called into action under if num === answer

## 4. Interactive Feedback
I added small visual improvements that make the game feel more responsive and fun to use, including:
- Button hover effects
- Cleaner message display
- Improved spacing and readability especially with the font

## Why These Features Improve the Game
These changes make the game more engaging, visually appealing, and fun to play. Instead of a basic layout, the game now feels more like a polished app with interactive feedback and a clear design style.
